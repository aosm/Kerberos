Kerberos for Macintosh News

New version of Kerberos for Macintosh ships with Mac OS X 10.3 (Panther)

October 24, 2003

The MIT Kerberos Development team is pleased to announce that as part of its ongoing collaboration with Apple, Mac OS X 10.3 (Panther) ships with another new version of Kerberos for Macintosh pre-installed.

Improvements in Kerberos for Macintosh 5.0 include a redesigned and streamlined Kerberos Login dialog with GUI for getting addressless and renewable tickets, krb5 1.3.1, KDC binaries and utilities now included, support for DNS lookups of KDCs, improved Windows interoperability, and an updated Kerberos application with improved appearance and performance and support for new KfM features.

For more details about changes and new features in Kerberos for Mac OS X, please see the Kerberos for Macintosh 5.0 release notes.

Mac OS X Kerberos Extras updated for Mac OS X 10.3 (Panther)

June 30, 2003

The Mac OS X Kerberos Extras package has been updated with support for Mac OS X Panther as well as continuing to support Mac OS X 10.2. This updated version is now available for downloading.

The Mac OS X Kerberos Extras install the Kerberos CFM support library, a sample Kerberos configuration file, and place an alias to the GUI Kerberos management application in /Applications/Utilities .

For more information and to download, see the Mac OS X Kerberos Extras web page. This release requires a Macintosh computer with Mac OS X 10.2 (Jaguar) or Mac OS X Panther.

Kerberos for Macintosh 4.0.3 now available

September 26, 2002

Kerberos for Macintosh 4.0.3, a maintenance update for both Mac OS X 10.1 and Mac OS 8 & 9, has been released and is available for downloading. Fixes include Classic ticket sharing compatibility with Mac OS X 10.2 (Jaguar) Kerberos. We recommend that all 4.0 - 4.0.2 users upgrade.

Instructions for downloading Kerberos for Macintosh 4.0.3 are on the Getting Kerberos for Macintosh page. This release requires a Macintosh computer with Mac OS X 10.1.2 or later version of 10.1 (Mac OS X components are not compatible with Mac OS X 10.2), or Mac OS 8.1 through 9.2.x for the Classic Mac OS components. A list of all the changes is available on the Kerberos for Macintosh 4.0.3 release page.

Mac OS X 10.2 includes new version of Kerberos for Macintosh

August 23, 2002

The MIT MacDev team is pleased to announce that as a result of its continuing collaboration with Apple, Mac OS X 10.2 ("Jaguar") ships with a improved version of Kerberos for Macintosh pre-installed.

Improvements include faster loading times, reduced memory usage, the first version of a supportable v5 API, krb5 1.2.5, krb524 support, new kswitch command-line utility for switching default users, and the GUI Kerberos application is now included.

Kerberos for Macintosh 4.0 now available

February 22, 2002

The final release of Kerberos for Macintosh 4.0 has been released and is available for downloading. This is the next major version of Kerberos for Macintosh, providing support for Mac OS X 10.1.2 and later. The 4.0 release expands and improves native support for Mac OS X, ticket sharing with the Classic environment, as well as including the new UI elements for Classic Mac OS introduced in KfM 3.5.

Instructions for downloading Kerberos for Macintosh 4.0 are on the Getting Kerberos for Macintosh page. This release requires a Power Macintosh with Mac OS X 10.1.2 or later version of 10.1, or Mac OS 8.1 through 9.2.2 for the Classic Mac OS components. Mac OS X 10.1 and earlier and Public Beta are no longer supported. A list of all the changes is available on the Kerberos for Macintosh 4.0 release page.

Mac OS X 10.1 includes Kerberos for Macintosh

September 26, 2001

The MIT MacDev team is pleased to announce that as a result of its collaboration with Apple, Mac OS X 10.1 ships with a version of Kerberos for Macintosh pre-installed.

The Kerberos included with Mac OS X 10.1 is a development version, essentially equivalent to the Kerberos for Macintosh 4.0a18 release. Mac OS X 10.1 Kerberos includes the Kerberos framework, user-level command line utilities, and a pre-release version of the Kerberos login authenticator.

For more information, see the Kerberos on Mac OS X FAQ page.

Kerberos for Macintosh 3.5 now available

June 29, 2001

Kerberos for Macintosh 3.5 has been released and is available for downloading. This is a major new release of Kerberos for Macintosh for Mac OS 8.x and 9.x. The 3.5 release incorporates the new UI elements for Classic Mac OS originally introduced in Kerberos for Macintosh 4.0 and Carbon versions of the KfM libraries. Many of the components have changed names to fit into future designs better.

We have created this new release in order to provide you with a new, final-quality Kerberos for Macintosh release in a timely manner. We expect KfM 4.0 development to continue for some time, mostly due to the Mac OS X work. The Classic Mac OS components are currently very stable, so we felt it was a good time to make them available separately.

Instructions for downloading Kerberos for Macintosh 3.5 are on the Getting Kerberos for Macintosh page. This release requires a Power Macintosh with Mac OS 8.1 through 9.1. It does not support Mac OS X, although it will work under Classic emulation. A list of all the changes is available on the Kerberos for Macintosh 3.5 release page.

MIT in collaboration with Apple to bring Kerberos to Mac OS X

May 18, 2000

The MIT MacDev team is pleased to announce a collaboration with Apple to bring Kerberos v4 and v5, including the KClient and GSS API's to Mac OS X. This upcoming Kerberos for Macintosh release will be available at the time of Mac OS X 1.0's release. Notification of the public testing releases of this package will be made on this web page and the Kerberos e-mail lists. Further information about this exciting development is available from the official MIT press release.
