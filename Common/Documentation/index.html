Kerberos for Macintosh
Kerberos for Macintosh is the implementation of the Kerberos authentication system for Mac OS X. It provides Kerberos v5 and Kerberos v4 protocols, GSSAPI, a graphical Login interface and accompanying API for acquiring Kerberos tickets, an in-memory ticket cache and KClient compatibility.

The current release of Kerberos for Macintosh is 5.0, included with Mac OS X 10.3.

You may need the Mac OS X Kerberos Extras for some applications to work properly.

The pages below mostly apply only to Mac OS X 10.2 and 10.3. For releases of Kerberos for Macintosh for earlier version of Mac OS X and Mac OS, see here.
