Kerberos for Macintosh Documentation

The documentation here applies to Mac OS X 10.2 and 10.3. For documentation for earlier Mac OS versions, go here.

Which component of Kerberos for Macintosh you would like information about?

Kerberos.framework

The framework which provides Kerberos and KClient APIs and services on Mac OS X.
Kerberos Configuration File (edu.mit.Kerberos)

The file where the Kerberos configuration is stored. Includes an explanation of how to configure Kerberos for Macintosh for your site!

Kerberos Application/Control Panel

The ticket and preferences management application/control panel.
Kerberos for Macintosh Installer

Information and customizable source for an MIT Kerberos for Macintosh installer.
