Kerberos Login API
What is the Kerberos Login API?

The Kerberos Login API provides a set of functions for simultaneously acquiring Kerberos v4 and/or Kerberos v5 ticket-granting tickets. This API is used by Kerberos for Macintosh to provide a consistent "single sign-on" user login experience and to enable utility applications to control Kerberos tickets and ticket options.

Where is the API specification?

Please refer to the Kerberos Login API for a list of the available functions and how to use them.

You can also find information on the Login and Logout Plugin API and Principal Translation Plugin API here.

Where do I get the Kerberos Login API?

The Kerberos Login API ships as part of Mac OS X 10.1.x and higher. CFM callers will also need the MIT Kerberos Extras to get CFM support libraries. You can find a link to the MIT Kerberos Extras here.

Please read the Kerberos Login API Bugs List and Version History for information on the latest release of the Kerberos Login API.
