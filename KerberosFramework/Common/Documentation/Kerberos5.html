Kerberos v5 in KfM

This documentation only applies to Mac OS X 10.2 and 10.3.

Kerberos for Macintosh includes an implementation of the Kerberos v5 application and server protocols.

Please refer to the papers and documentation describing the Kerberos v5 protocol and MIT implementation listed here.

You can see what Kerberos v5 functions are exported in Kerberos for Macintosh by consulting /System/Library/Frameworks/Kerberos.framework/Headers/krb5.h .

The latest release is included with Mac OS X 10.2 and later.
