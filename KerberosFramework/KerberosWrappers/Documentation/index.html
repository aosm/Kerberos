KerberosWrappersLib
What is KerberosWrappersLib?

KerberosWrappersLib is a CFM shared library for use with MIT Kerberos for Macintosh. It provides C++ classes for accessing Kerberos contexts, principals and credentials.

Where is the API specification?

Please refer to the KerberosWrappersLib API for a list of the available functions and how to use them.

Where do I get KerberosWrappersLib?

The KerberosWrappersLib SDK is part of MIT Kerberos for Macintosh. You can get the latest release of the MIT Kerberos for Macintosh here.

Please read the KerberosWrappersLib Bugs List and Version History for information on the latest release of KerberosWrappersLib.
