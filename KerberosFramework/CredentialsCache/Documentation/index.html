Credentials Cache API
What is the Credentials Cache API?

The Credentials Cache API provides an API for accessing the in-memory credentials cache for Kerberos v4 and Kerberos v5 tickets. The Credentials Cache API is used by other libraries in the Kerberos framework and should not normally be needed by Kerberos applications.

Where is the API specification?

There are two versions of the Credentials Cache API available to callers. The most recent API is Credentials Cahce API v5, although the most recent documentation available is for Credentials Cache API v3. (For the most part, functions have only been added, not changed or removed, since then.) Credentials Cache API v2 is available for compatibility.

Where do I get the Credentials Cache API?
An implementation of the Credentials Cache API is included with Kerberos for Macintosh. The latest release is included with Mac OS X 10.2 and later.

Please read the Credentials Cache library Bugs List and Version History for information on the latest release of the Credentials Cache library.
